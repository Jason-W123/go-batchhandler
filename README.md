## Go-batchHander
Note: It's still under development, not ready for public use.

### Import nitro
Clone this repo:
```bash
git clone https://github.com/Jason-W123/go-batchhandler.git
cd go-batchhandler
```

Clone submodules:
```bash
git submodule update --init --recursive --force
```

Enter nitro:
```bash
cd nitro
```

Build necessary libs:
```bash
make contracts
make build-node-deps
```
